# Scoring Dimensions (1-10 each)

1. **structural_depth**: Does this scenario change basic operating models or value chain (not just a surface-level adaptation)?
   - 8-10 = Rewires the basic operating model or value chain
   - 4-7 = Significant but partial structural change
   - 1-3 = Incremental change; existing model largely survives

2. **irreversibility**: Once this change is implemented, would it be too costly or practically impossible to return to previous models or operational systems?
   - 8-10 = Reversal is prohibitively costly or practically impossible
   - 4-7 = Reversal would require enormous resources
   - 1-3 = Could realistically be undone

3. **industry_related**: Does this scenario directly affect core industry processes of the target industries ({target_industries})?
   - 8-10 = Directly affects core industry processes / value chain
   - 4-7 = Adjacent to or partially affects core processes
   - 1-3 = Tangential or unrelated to the target industries' core processes
   - Score this independently of topic — focus only on industry fit.

4. **topic_relevance**: How directly does this scenario relate to the project topic "{topic}"?
   - 8-10 = Fundamentally about the topic; topic is the core driver
   - 4-7 = Topic is a major angle but not the only driver
   - 1-3 = Topic is tangential or absent
   - Score this independently of industry — focus only on topic fit.

5. **feasibility**: Could this scenario realistically happen within the next 10-15 years?
   - 8-10 = Strong evidence it is already underway or highly likely; clear enabling trends exist
   - 4-7 = Plausible but requires several conditions to align; some enabling trends visible
   - 1-3 = Speculative; requires breakthrough events or unlikely preconditions

# IMPORTANT SCORING RULES
- total_score MUST equal structural_depth + irreversibility + industry_related + topic_relevance + feasibility (max 50).
