# Scoring Dimensions (1-10 each)

1. **structural_depth**: Does this scenario describe a change to the basic operating model or value chain — not just a surface-level adaptation?
   - 8-10 = Rewires how the industry fundamentally operates
   - 4-7 = Significant but partial structural change
   - 1-3 = Incremental change; existing model largely survives

2. **irreversibility**: Once this change takes hold, how practically impossible is it to return to the previous model?
   - 8-10 = Reversal is prohibitively costly or physically impossible
   - 4-7 = Reversal would require enormous resources
   - 1-3 = Could realistically be undone

3. **industry_relevance** (OPTIONAL — score even if threshold is 0, for reference):
   How directly and centrally does this affect core processes in the target industries?
   - 8-10 = Central to the target industries' core processes
   - 4-7 = Affects adjacent parts of the industries
   - 1-3 = Tangential or indirect connection

4. **topic_relevance** (OPTIONAL — score even if threshold is 0, for reference):
   How directly does this scenario relate to the core topic "{topic}"?
   - 8-10 = The scenario is fundamentally about this topic; removing the topic would make the scenario meaningless
   - 4-7 = The topic is one of several drivers; the scenario touches the topic but is not centered on it
   - 1-3 = The topic is barely relevant; the scenario could exist without this topic

5. **feasibility**: Could this scenario realistically happen within the next 10-15 years?
   - 8-10 = Strong evidence it is already underway or highly likely; clear enabling trends exist
   - 4-7 = Plausible but requires several conditions to align; some enabling trends visible
   - 1-3 = Speculative; requires breakthrough events or unlikely preconditions

# IMPORTANT SCORING RULES
- total_score MUST equal structural_depth + irreversibility + industry_relevance + topic_relevance + feasibility (max 50).
